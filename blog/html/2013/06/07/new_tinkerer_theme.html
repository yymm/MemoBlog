Tinkerer 1.2 Release!

Tinkererのテーマをいじろうかどうか迷っていた矢先にバージョンアップが。

しかも新テーマflatが。

Home — Tinkerer かっけぇ。

シンプルで良い感じ。

即conf.pyをいじった。

pipでパッケージを更新するのはなにげに初めてだった。

$ workon Tinker
$ pip install tinkerer -U
$ vim conf.py
dark -> flat
$ tinker -b
$ hg add
$ hg ci -m "use flat!"
$ hg push

pip updateかあるのかと思ったけど、-U(or –upgrade)オプションを付ければいい。
